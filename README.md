# 广西区域化岗位数据供需一体化平台

本仓库发布广西区域化岗位数据供需一体化平台交互原型，面向学校、学院、工信、人社、就业处和学生等角色，提供就业态势、结构诊断、广西区域查询、产业协同、运营作战、行动预警和 AI 数据透视能力。

## 在线访问

GitHub Pages: <https://alinagong-design.github.io/gx-job-supply-demand-ai-cockpit/>

## 核心功能

- 2025届就业核心指标与毕业去向结构
- 学院、群体、区域和产业结构分析
- 广西14个设区市数据球查询入口
- 地区上下文智能问答与可信依据展示
- 学院诊断、产业协同和就业运营专题
- 行动预警、研判任务和数据口径说明
- 右下角 AI 数据透视快捷入口

## 数据说明

原型中的全区正式指标与城市参考标签分区展示。城市级就业人数、岗位数量、薪资、技能和供需匹配率在数据接入前不会生成推测性结果。
